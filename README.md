# Netflix Clone Project

# Description
A responsive Netflix clone built using HTML and CSS.

# Features
- Responsive design for various screen sizes
- Replicates the Netflix homepage layout and design

# Technologies Used
- HTML5
- CSS3

- styles.css: The CSS stylesheet
